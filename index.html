DSMovie
Neste evento foi desenvolvido um projeto de site para avaliação de filmes. Foram 3 episódios, do zero à implantação na Nuvem:
1. Front End estátio
Criação do Projeto, repositório no GitHub, layout estático, Bootstrap, componentes React, HTML e CSS, rotas e navegação.
2. Back End
Modelo de domínio, database seed, busca paginada, salvar dados, API REST, perfif de teste com banco H2, perfil de homologação e produção com banco Postgresql (eu utilizei o Insomnia).
3. Integração e Implementação
Props, requisição com Axios, React Hooks, parâmetros de rota, integração front end com back end e implementação na Nuvem.
